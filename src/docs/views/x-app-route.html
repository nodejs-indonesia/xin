App, views, and routing

Xin has built-in multi views application model. All you have to do is using xin-app, xin-view and xin-pager.

Below is example to create application with multiple views. 

<xin-app>
  <template>
    <div>
      <a href="#!/">Home</a>
      <a href="#!/foo">Foo</a>
      <a href="#!/bar">Bar</a>
    </div>

    <xin-pager>
      <xin-view uri="/">
        <template>
          This is home
        </template>
      </xin-view>

      <xin-view uri="/foo">
        <template>
          This is foo
        </template>
      </xin-view>

      <xin-view uri="/bar">
        <template>
          This is bar
        </template>
      </xin-view>
    </xin-pager>
  </template>
</xin-app>
